# chatgpt-python-gui

### Super simple Chat-GPT-GUI

#### Needed Python Module:
```
pip3 install openai
```
#### Generate API-Key:
```
https://beta.openai.com/account/api-keys
```
#### Command:
```
python3 chatGTP.py
```
